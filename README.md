## What You Manage — Admin Panel

> **No code access required after setup.**

| Area | Capabilities |
|:--|:--|
| **AI Models** | Enable/disable models, set credit cost |
| **API Keys** | OpenAI, Gemini, Kling — managed in UI |
| **Pricing** | Create plans and credit packages |
| **Users** | Roles, credits, access control |
| **CMS** | Landing page, testimonials, legal pages |
| **Features** | Toggle features without redeploy |
| **Analytics** | Revenue, usage, system health |
